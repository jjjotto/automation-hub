# Automation Hub Documentation

This directory contains documentation and visual assets for the Automation Hub application.

This screenshot shows the complete HTML mockup page including:

### Application Window
**File:** `automation-hub-window-only.png`

This screenshot shows just the application window itself, displaying:
- Window title bar with minimize, maximize, and close buttons
- "Automation Hub Jobs" header

## Note

The Automation Hub is a WPF (.NET 8) desktop application that requires Windows to run. These screenshots and mockups were created to demonstrate the application's appearance and functionality without requiring a Windows environment.

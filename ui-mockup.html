How to Run the Actual Application

System Requirements:

  • Windows 10 (22H2+) or Windows 11
  • .NET 8.0 Desktop Runtime or SDK
  • Access to Y:\temporary_files\JO\automation (or local-config directory)

To run the application:

  1. Clone the repository: git clone https://github.com/jjjotto/automation-hub.git
  2. Open in Visual Studio 2022 or use command line
  3. Build: dotnet build
  4. Run: dotnet run --project AutomationHub.App/AutomationHub.App.csproj

Configuration:

The application automatically falls back to a local-config directory if the Y: network drive is not available, making it easy to run in development mode.

For detailed instructions, see RUNNING_THE_APP.md and UI_SCREENSHOTS.md
